Get running in 60 seconds

1
Download the .dmg
Choose Apple Silicon (arm64) for M1/M2/M3 Macs, or Intel (x64) for older Macs.
2
Open the installer
Double-click the .dmg file. Drag FreeTorrent into your Applications folder.
3
Launch FreeTorrent
Open Launchpad or Applications and click FreeTorrent. On first launch macOS may ask you to confirm — click Open.
4
Add a torrent
Click "Add Torrent", choose a .torrent file, and the download begins automatically.
1
Download the installer
Download FreeTorrent-Setup-x64.exe for a full install, or the Portable .exe to run without installing.
2
Run the installer
Double-click the .exe. Windows SmartScreen may warn you — click "More info" then "Run anyway". FreeTorrent is open source and safe.
3
Launch from Start Menu
Find FreeTorrent in the Start Menu or on your Desktop shortcut.
1
Download the AppImage
The AppImage works on any Linux distro without installation.
2
Make it executable
chmod +x FreeTorrent-*.AppImage
3
Run it
./FreeTorrent-*.AppImage
4
Or install the .deb (Ubuntu/Debian)
sudo dpkg -i FreeTorrent-amd64.deb
// what's inside

Built different

No libtorrent. No shortcuts. Every protocol byte written by hand.

🔐
SHA-1 Verification
Every piece is hashed and verified before writing to disk. Corrupt data is automatically discarded and re-downloaded.
🔄
Resume Downloads
Restart anytime. Existing pieces are SHA-1 verified on startup and only clean data is skipped.
📁
Multi-file Torrents
Correctly maps pieces across file boundaries. Handles any torrent layout — single file or complex multi-directory structures.
⬆️
Seeds While Downloading
Starts seeding verified pieces immediately. Continues seeding automatically after the download completes.
🌐
UDP + HTTP Trackers
Speaks both tracker protocols. Falls back to HTTP automatically when UDP is blocked by your network.
⚡
Zero Bloat
No ads, no auto-updates calling home, no telemetry. The engine is ~7 files of plain Node.js. Read every line on GitHub.
